MultiAgentPathFinding.build_weights_matrix — Function
build_edge_weights_matrix(mapf[, edge_weights_vec])

Turn a vector edge_weights_vec into a sparse weighted adjacency matrix for the graph mapf.g.

This function doesn't allocate because the necessary index information is already in the MAPF object.
